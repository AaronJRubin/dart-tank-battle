About this Game

This is a simple tank shooting game, implemented in the Dart programming language and designed to be played by multiple players around a single keyboard.

The controls were designed to minimize the amount of keys that would need to be processed simultaneously. No more than two keys should be pressed at the same time per player! This concern motivated my decision to make shooting automatic, rather than player-controlled.

Most keyboards only allow for up to six key-presses to be registered at once. Consequently, playing this game with four players, which would require up to eight simultaneous key-presses, is not recommended unless you have a very high-end keyboard.

Incidentally, the ability of a keyboard to handle an arbitrary number of key presses at once is called n-key rollover. Look it up if you're interested!

Technologies used in this game include three.dart (for the game itself), HTML5 local storage for game settings, and polymer, including the recently released Paper elements, for various user interface elements. Because HTML5 local storage is used, the game settings on this page may not work on very old browsers, forcing you to play with defaults. In addition, Polymer, the web components technology on which it is based, and Paper elements are new technologies that may not work across all browsers, even modern ones. Chrome is best, Firefox is fine (a bit laggy), and Safari isn't quite there yet.
